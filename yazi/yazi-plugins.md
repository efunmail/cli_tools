## FG - (Fzf, RipGrep, Bat)

- Repo: https://gitee.com/DreamMaoMao/fg.yazi

> Dependencies
> - fzf
> - ripgrep
> - bat

- Install:

```sh

git clone --depth 1 https://gitee.com/DreamMaoMao/fg.yazi ~/.config/yazi/plugins/fg.yazi
```
